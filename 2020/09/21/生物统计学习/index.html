生物统计学习

发表于 2020-09-21

生物统计是一门重要的学科，能够透彻理解统计知识，是科研道路的重大助力。

常用术语

变量 (variable)

是指某种特征，其表现随个体而异。

  • 身高、体重、体长、产奶量、毛色
  • 随机变量：变量取值是变化的是不可预测的

观测值 (observation)

对变量进行测量和u哦这观察所获得的数值，也称变数(variate)。

总体 (population)

一个统计问题所研究对象的全体，它是具有某种或某些共同特征的元素的集合。

  • 总体的分类：
  1. 总体中所含单元数目的有限性：有限总体和无限总体
  2. 总体中的构成是否随时间的变化而变化：动态总体和固定总体
    1. 总体是否由现实存在的研究对象所构成：现实总体和假想总体

      个体 (individual)

      组成总体的每一个基本单元。

样本 (sample)

从总体中按一定方法抽取部分具有代表性的个体，这部分个体称为样本。

参数

描述总体特征的数，通常是未知的，总体平均数，总体方差。

统计量

描述样本特征的数，是样本观测值的已知函数。

  • 样本平均数
  • 样本方差

准确性

观测值或估计值与真实值的接近程度

精确性

重复观测值或估计值之间的接近程度（与数据的有效位数有关）

生统的主要内容

资料分类和异常值处理

从上述数据中很难看出平均水平和变异程度，所以我们首先需要对资料进行描述性统计分析：

  1. 数据资料的审核与修订
  2. 统计资料的分组和汇总（内部结构、类型、特征）
  3. 原始资料或汇总资料的各种基本特征计算。（集中趋势和离散趋势）
  4. 用统计表或者统计图等简单形式直观反映资料的基本特征和变化趋势。

数据整理


资料的分类

  • 连续性资料
    在一定范围内可以取任何实数值的数据资料。
    （体重、产奶量、产毛量）

  • 离散性资料
    在一定范围内只取有限种可能值得数据资料。
    计数资料：产仔数、产蛋数
    分类资料：性别、基因型、毛色

  • 定量资料
    是体现个体间在量上差异的资料，上述连续性资料和计数资料都是定量资料。

异常数据的判断

  • 异常数据
    指定量资料中的极端值，其数值明显偏离它（或它们）所属样本的其余观测值。

  • 存在两种情况

    1. 变量内在随机变异性的极端表现————正常（保留）
    2. 试验或数据记录过程出现错误或试验条件异常所导致————异常（舍弃或修正）

异常数据保留/舍弃判断方法

  • 检查数据采集过程
  • 检查发生该数据的个体
  • 统计学方法检验：
  1. 四分位数检验
  2. 格拉布斯检验

百分位数（Quartile）

把一组数据从小到大排列，如果小于某数值的数据个数为全体数据个数的x%，则称该数为第x百分位数，或x%分位数。

x%分位数计算：
n个数从小到大排列，令 i=n x%
若i不是整数，大于i的毗邻整数为x%分位数的位置
若i是整数，x%分位数是第i个数和第（i+1）个数的平均数

第25百分位数又称为下四分位数（LQ，Q1），第50百分位数又称中位数（median），第75百分位数又称上四分位数（UQ，Q3）。

四分位数检验

  1. 将所有数据大小排序
  2. 计算上四分位数UQ（Q3）
  3. 计算下四分位数LQ（Q1）
  4. 计算中四分位数范围（IQR）

    IQR = UQ - LQ

所有大于UQ+1.5IQR或者小于LQ-1.5IQR的数据都可判定为异常数据。

举个例子

有数据1，2，3，4，5，6，7，8，这8个数据，试判断这8个数据中有无异常数据？

根据定义：

  1. UQ=7
  2. LQ=3
  3. IQR=7-3=4
  4. UQ+1.5IQR=7+6=13
  5. LQ-1.5IQR=3-6=-3

示例数据中没有大于13的数也没有小于-3的数，所以该组数据没有异常值。

格拉布斯检验（Grubbs）

  • 适用范围
    用于来自正态总体样本的异常值检验。

  • 检验步骤

  1. 计算T值：T=（|算术平均数 - Xi|）/ 样本标准差
  2. T值与表中临界值进行比较
    1. 比较结果分析
      若 T值 > 临界值——Xv为异常值，剔除Xv，检验下一个极端值，直到没有异常值为止。
      临界值与样本容量n和显著水平a有关。
举个例子

有数据1，2，3，4，5，6，7，8，这8个数据，试判断这8个数据中有无异常数据？

计算得：

  • 样本平均数 = 4.5 ，样本标准差 = 2.45
  • Xi=1时，T=1.43，样本容量n=8，查表得T0.05=2.13>T；同理，Xi=8时，T=1.43，T0.05=2.13>T，所以该组数据中没有异常值。

数据得频数（频率）分布

  • 频率分布（frequency distribution）
    反映一组数据中不同观测值出现的频率（频数），可以直观地反映出一组数据的分布特征和趋势。
  • 离散性资料的频率分布
    计数资料和分类资料的频率分布：每个自然值或属于不同类别的观测值出现的次数（频数）和所占的比例。
  • 连续性资料的频率分布
    1. 求全距：变数的最大值和最小值之差
    2. 确定组数：取决于样本含量
    3. 确定组距：每组的最大值与最小值之差——组距=全距/组数 （组距为整数，各组的组距相同，用i表示）
    4. 求组中值和组限
    5. 根据组限，将资料中的数据分别分配到各组中
    6. 计算各组的频率和频数

举个例子

  1. 求全距：R=19.3-8.5=10.8kg
  2. 确定组数：n=200，由上表确定为12组
  3. 确定组距：组距（i）=全距/组数
    i=10.8/12=1.0
  4. 求组中值和组限
    组中值=组下限+1/2组距
    第一组：组下限=组中值-1/2组距=8.0 ； 组上限=8.0+1.0=9.0
  5. 归组划线计数
  6. 统计频数和频率，完成频数分布表
