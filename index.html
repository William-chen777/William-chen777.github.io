Python的正则表达式

发表于 2020-03-31 | 分类于 Python

虽然在 Python 中使用正则表达式有几个步骤，但是每一步都相当简单

  1. 用 import re 导入正则表达式模块
  2. 用 re.compile()函数创建一个Regex对象（原始字符）
  3. 向Regex对象的search()方法传入想要查找的字符串，它将返回一个Match对象
  4. 调用Match对象的group()方法,返回实际匹配文本的字符串

    举例说明：

    phone_number_regex = re.compile(r'\d\d\d-\d\d\d-\d\d\d\d')
mo = phone_number_regex.search('My number is 415-555-5555.')
print ('Phone number found: ' + mo.group())
>>>Phone number found: 415-555-5555
