sklearn-DecisionTreeClassifier的基本用法

决策树的流程

计算全部特征的不纯度，选取不纯度指标最优的特征来分支，在第一个特征的分支下，计算全部特征的不纯度指标，选取不纯度指标最优的特征继续分支

DecisionTreeClassifier的核心参数：criterion

为了要将表格转化为一棵树，决策树需要找出最佳节点和最佳的分枝方法，对分类树来说，衡量这个“最佳”的指标
叫做“不纯度”。通常来说，不纯度越低，决策树对训练集的拟合越好。现在使用的决策树算法在分枝方法上的核心
大多是围绕在对某个不纯度相关指标的最优化上。

  • entropy 信息熵
  • gini 基尼系数

Python的正则表达式

发表于 2020-03-31 | 分类于 Python

虽然在 Python 中使用正则表达式有几个步骤，但是每一步都相当简单

  1. 用 import re 导入正则表达式模块
  2. 用 re.compile()函数创建一个Regex对象（原始字符）
  3. 向Regex对象的search()方法传入想要查找的字符串，它将返回一个Match对象
  4. 调用Match对象的group()方法,返回实际匹配文本的字符串

    举例说明：

###正则表达式的符号

  1. ?匹配零次或一次前面的分组
  2. *匹配零次或多次前面的分组
  3. +匹配一次或多次前面的分组
  4. {n}匹配n次前面的分组
  5. {n,}匹配n次到更多前面的分组
  6. {,m}匹配零次到m次前面的分组
  7. {n,m}匹配至少n次、最多m次前面的分组
  8. {n,m}?或*?或+? 对前面的分组进行非贪心匹配
  9. ^spam 意味着字符串必须从spam开始
  10. spam$意味着字符串必须以spam结束
  11. .匹配所有字符，换行符除外
  12. \d, \w, \s分别匹配数字，字母和空格
  13. \D, \W, \S分别匹配除了数字，字母和空格外的所有字符
  14. [abc]匹配括号内任意字符(如a，b或c)
  15. [^abc]匹配不在括号内的任意字符

###关于正则表达式的18个问题
1.创建Regex对象的函数是什么？

re.compile()返回Regex对象

===

2.在创建Regex对象时，为什么常用原始字符串？

使用原始字符串是为了让反斜杠不必转义

===

3.search()方法返回了什么？

返回Match的对象

===

4.通过Match对象，如何得到匹配该模式的实际字符串？

group()方法返回匹配文本的字符串

===

5.用r’(\d\d\d)-(\d\d\d-\d\d\d\d)’创建的正则表达式中，分组0表示什么？分组1呢？分组2呢？

分组0是整个匹配，分组1 是匹配第一个括号，分组2匹配第二个括号

===

6.括号和句点在正则表达式语法中有特殊的含义，如何制定正则表达式匹配真正的括号和句点字符?

句号和括号可以用反斜杠转义：.; ()

===

7.findall()方法返回一个字符串的列表。是什么决定它提供哪种返回？

如果正则表达式没有分组，就返回字符串的列表。如果正则表达式有分组，就返回字符串的元组列表。

===

8.在正则表达式中，|字符表示什么意思？

匹配两个组中的任何一个，是“或”的意思

===

9.在正则表达式中，?字符有哪两种含义？

?字符可以表示“匹配前面的分组0次或1次”， 或者用于表示非贪心匹配（贪婪匹配）

===

10.在正则表达式中，+和*字符之间的区别是什么？

+匹配1次或者多次； *匹配0次或者多次

===

11.如何让正则表达式不区分大小写？

将re.I或者re.IGNORECASE作为第二个参数传入re.compile()

===

12.字符.通常匹配什么？如果re.DOTALL作为第二个参数传递给re.compile(),会匹配什么？

字符.匹配任何字符，换行符除外，如果传入该参数，那么.也会匹配换行符

===

13..*和.?的区别是什么？

.执行贪心匹配，.?执行非贪心匹配

===

14.匹配所有数字和小写字母的分类语法是什么？

[0-9a-z]|[a-z0-9]

===

15.如果num_regex = re.compile(r'\d+'),那么num_regex.sub('X', '12 drummers, 11 pipers, five rings, 3 hens')返回什么？

‘X drummers, X pipers, five rings, X hens’

===

16.写一个正则表达式，匹配每3位就有一个逗号的数字。
它必须匹配:‘42’， ‘1，234’， ‘6，368，745’
但是不会匹配：‘12，34，567’， ‘1234’

re.compile(r'^\d{1,3}(,\d{3})*$') #必须以数字开头，开头数字由1-3个组成，接着由若干个‘，’ + 3个数字 组成，且以此结尾

===

17.写一个表达式，匹配姓为‘Nakamoto’的完整姓名。可以假定名字总是出现在姓的前面，是一个大写字母开头的单词。
它必须匹配：‘Satoshi Nakamoto’， ‘Alice Nakamoto’
但是不会匹配：‘satoshi Nakamoto’， ‘Mr.Nakamoto’， ‘Satoshi nakamoto’

re.compile(r'[A-Z][a-z]*\sNakamoto')

===

18.编写一个正则表达式匹配一个句子，第一个单词是Alice, Bob或者Carol， 第二个单词是eats, pets, throws, 第三个单词是apples, cats, baseballs，
该剧自以句点结束。这个正则表达式不区分大小写。

re.compile(r'(Alice|Bob|Carol)\s(eats|pets|throws)\s(apples|cats|baseballs)\.', re.IGNRECASE)
